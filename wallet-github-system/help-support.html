Frequently Asked Questions
How do I earn CODE tokens?
‚ñº
You earn CODE tokens by writing code, using AI agents, and contributing to repositories. Every line of code, function, class, and test you write is tracked and rewarded automatically.
How do I connect my wallet?
‚ñº
Click on "Wallet Settings" in the Settings menu. You'll need MetaMask installed. The system will automatically switch you to the Base network for optimal performance and low fees.
How do AI agents work with my code?
‚ñº
AI agents can read your repository requirements, write code, and push directly to your GitHub repositories. You maintain full control and can review all changes before they're merged.
Is my private code secure?
‚ñº
Yes! CodeDAO only tracks coding activity metrics (lines, functions, etc.) - never your actual code content. Private repositories remain private, and you control all access permissions.
What networks are supported?
‚ñº
CodeDAO primarily operates on Base network for fast, low-cost transactions. We also support Ethereum mainnet, but recommend Base for the best user experience.

Common Issues & Solutions

Wallet Connection Issues:

  1. 1
    Make sure MetaMask is installed and unlocked
  2. 2
    Check that you're on the correct network (Base or Ethereum)
  3. 3
    Clear browser cache and refresh the page
  4. 4
    Try disconnecting and reconnecting your wallet

AI Agent Problems:

  1. 1
    Check that your GitHub account is properly linked
  2. 2
    Verify API keys are correctly configured
  3. 3
    Ensure repository permissions allow AI access
  4. 4
    Check the AI Agent preferences settings
