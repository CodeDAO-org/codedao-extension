‚ùì Frequently Asked Questions

üíª What kind of code counts for rewards?
Any code committed to a Git repository! This includes bug fixes, new features, documentation, tests, refactoring, and even configuration files. Quality matters more than quantity.
üìà How is code quality measured?
Our algorithm analyzes multiple factors: presence of tests, code documentation, file complexity, commit frequency, and contribution scope. Better practices = higher multipliers.
üéØ Can I use my existing projects?
Absolutely! CodeDAO works with any Git repository - personal projects, open source contributions, work projects (with permission), or brand new experiments.
üí∞ Are CODE tokens real cryptocurrency?
Yes! CODE is an ERC-20 token deployed on Base network. You can transfer, trade, or hold them like any other cryptocurrency. Contract: 0x1F8b43F7aeD0D1b524Ec5b4930C19098E8D4fbD0
üîí Is my code data private?
CodeDAO only analyzes commit metadata (lines changed, file types, etc.) - never your actual source code. Your intellectual property remains completely private.
‚ö° How often can I claim rewards?
Anytime! Rewards accumulate automatically and you can claim them whenever you want. There's no minimum threshold or waiting period.
